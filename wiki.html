Academic Positions(ตำแหน่งทางวิชาการ)

Lists of Academic Position

แสดงข้อมูลรายการตำแหน่งทางวิชาการ

  • URL   https://mis-api.cmu.ac.th/ref/api/academic-positions
  • Method   GET
  • URL Params
    Name Type Required Default Description
    q String ค้นหาตำแหน่งทางวิชาการ

  • Responses

    • Status Code   200
    • Content-Type   application/json

    • Content   [Array] 

        [
    {
      id: 3,  // (int) PK
      code: "02",  // (String) รหัสจาก สกอ.
      name_th: "ผู้ช่วยศาสตราจารย์",  // (String) ชื่อเต็มภาษาไทย
      name_en: "Assistant Professor",  // (String) ชื่อเต็มภาษาอังกฤษ
      title_th: "ผศ.",  // (String) ชื่อย่อภาษาไทย
      title_en: "Asst. Prof.",  // (String) ชื่อย่อภาษาอังกฤษ
      description: "ตำแหน่งทางวิชาการ"  // (String) คำอธิบายเพิ่มเติม
    },...,
  ]
